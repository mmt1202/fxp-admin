# 房小评 Admin

房小评后台管理前端，用于管理用户、房源、内容审核、运营、系统配置等后台模块。

## 模块联调状态

状态定义：

- **已联调**：页面已按后台接口接入真实数据或操作接口。
- **前端预览**：页面交互和展示已完成，但当前主要用于产品/运营预览。
- **等待后端**：页面已存在，但缺少可确认的后端接口联调，不应按完整完成模块验收。
- **Mock 数据**：页面当前依赖示例数据或本地模拟数据。
